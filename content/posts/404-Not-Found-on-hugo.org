* やったこと

easy-hugoでPreviewするとEmacsがカタマリました\\
hugo server後http://localhost:1313/
を確認すると、"404 not Found"となりました\\
上記、Webpageを参照に以下の手順で 404 Not Found を回避できました

1. pulbicフォルダの中身を削除
2. デプロイ後、hugo server
#+BEGIN_SRC sh
$ hugo -D
$ hugo server
#+END_SRC

上記でeasy-hugoでのPreviewもできる様になりました
